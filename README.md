# CraftTool

> Open-source development templates for everyone.

![CraftTool Banner](https://raw.githubusercontent.com/CraftTool/.github/main/assets/banner.png)

## Our Mission

CraftTool provides high-quality, reusable project templates covering the complete development workflow from initialization to deployment.

## Templates

| Template | Description | Tech |
|----------|-------------|------|
| [github-action-template](https://github.com/CraftTool/github-action-template) | GitHub Action dev template | TypeScript |
| [typescript-tool-template](https://github.com/CraftTool/typescript-tool-template) | TypeScript library template | TypeScript |
| [nextjs-blog-template](https://github.com/CraftTool/nextjs-blog-template) | Next.js blog template | Next.js |
| [flutter-app-template](https://github.com/CraftTool/flutter-app-template) | Flutter app template | Dart |
| [hexo-theme-template](https://github.com/CraftTool/hexo-theme-template) | Hexo theme template | EJS/SCSS |
| [hugo-theme-template](https://github.com/CraftTool/hugo-theme-template) | Hugo theme template | Go Templates |
| [python-tool-template](https://github.com/CraftTool/python-tool-template) | Python library template | Python |

## Quick Start

```bash
# Use GitHub CLI
gh repo create my-project --template CraftTool/typescript-tool-template
```
