# CraftTool

> 开箱即用的开发模板库，让你的项目从第一天就拥有最佳实践。

![CraftTool Banner](https://raw.githubusercontent.com/CraftTool/.github/main/assets/banner.png)

## ✨ 我们的使命

CraftTool 致力于为开发者提供**高质量、可复用**的项目模板，覆盖从初始化到部署的完整开发流程。

## 📦 模板仓库

| 模板 | 描述 | 技术栈 |
|------|------|--------|
| [github-action-template](https://github.com/CraftTool/github-action-template) | GitHub Action 开发模板 | TypeScript |
| [typescript-tool-template](https://github.com/CraftTool/typescript-tool-template) | TypeScript 工具库模板 | TypeScript |
| [nextjs-blog-template](https://github.com/CraftTool/nextjs-blog-template) | Next.js 博客模板 | Next.js / TypeScript |
| [flutter-app-template](https://github.com/CraftTool/flutter-app-template) | Flutter 应用模板 | Dart / Riverpod |
| [hexo-theme-template](https://github.com/CraftTool/hexo-theme-template) | Hexo 博客主题模板 | EJS / SCSS |
| [hugo-theme-template](https://github.com/CraftTool/hugo-theme-template) | Hugo 博客主题模板 | Go Templates |
| [python-tool-template](https://github.com/CraftTool/python-tool-template) | Python 工具库模板 | Python |

## 🚀 快速开始

### 使用模板创建新项目

1. 选择你需要的模板仓库
2. 点击 **"Use this template"** 按钮
3. 开始开发！

### 示例：创建新的 TypeScript 项目

```bash
# 使用 GitHub CLI
gh repo create my-new-project --template CraftTool/typescript-tool-template
```
